# Code Collaboration Platform - Frontend

CodeSync is a modern, real-time collaborative code editor designed for teams. It enables seamless pair programming and collaborative development with a rich set of features.

## ✨ Features

- **Real-time Collaboration**: Edit code with others in real-time.
- **Microservice Architecture**: Built with a scalable microservice backend.
- **Rich Editor Experience**: Support for multiple languages and themes.
- **Branch-based Versioning**: Navigate through code history using branches.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Ashish-Mishra11/Code-Collaboration-Platform-Frontend.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack

- **Frontend**: React, Vite, CSS (Vanilla)
- **State Management**: React Hooks
- **Communication**: WebSockets (SockJS, Stomp)
- **Styling**: Modern CSS with glassmorphism and dynamic animations.

## 📂 Project Structure

- `src/components`: Reusable UI components.
- `src/pages`: Main application pages (Editor, Dashboard, etc.).
- `src/services`: API and WebSocket service logic.
